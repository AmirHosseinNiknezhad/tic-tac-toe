
# Tic Tac Toe

A simple implementation of the classic Tic Tac Toe game.

## Features

- Interactive command-line interface
- AI plays perfectly using full game tree
- Ability to toggle visibility of position and all possible move evaluations
- Win/draw detection
- Input validation

## Installation

Clone the repository:
```bash
git clone https://github.com/yourusername/tic-tac-toe.git
cd tic-tac-toe
```

## Usage

Run the game:
```bash
python main.py
```

## Rules

- Players take turns marking spaces
- First player to get three marks in a row (horizontal, vertical, or diagonal) wins
- If all spaces are filled with no winner, the game is a draw

# Tic Tac Toe

A simple implementation of the classic Tic Tac Toe game.

## Features

- Interactive command-line interface
- AI opponent with perfect play using minimax algorithm
- Optional game analysis showing position evaluation and move scores
- Win/draw detection with input validation

## Installation

Clone the repository:
```bash
git clone https://github.com/yourusername/tic-tac-toe.git
cd tic-tac-toe
```

## Usage

Run the game:
```bash
python main.py
```

Alternatively, you can use UV to manage the environment:
```bash
uv run main.py
```

(UV is optional and not required to run the game)

## Rules

- Players take turns marking spaces
- First player to get three marks in a row (horizontal, vertical, or diagonal) wins
- If all spaces are filled with no winner, the game is a draw
